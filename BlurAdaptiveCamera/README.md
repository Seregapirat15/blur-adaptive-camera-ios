# Blur Adaptive Camera (iOS)

Нативное приложение на **SwiftUI** для iPhone: оценка **размытости** кадра по **дисперсии отклика фильтра Лапласа** на сетке, **теплокарта** «где резче / где размытее», **адаптивная** сила **`CISharpenLuminance`** (Core Image), опционально **рамки лиц** через **Vision**.

## Требования

- macOS с **Xcode 15+** (рекомендуется)
- **iPhone** с iOS **16+**, кабель USB / доверие к компьютеру
- Apple ID для подписи (бесплатный аккаунт подходит для установки на свой телефон)

## Как открыть и запустить

1. Скопируйте папку `BlurAdaptiveCamera` на Mac (через облако, флешку и т.д.).
2. Откройте **`BlurAdaptiveCamera.xcodeproj`** двойным щелчком.
3. В навигаторе слева выберите проект **BlurAdaptiveCamera** → target **BlurAdaptiveCamera** → вкладка **Signing & Capabilities**:
   - включите **Automatically manage signing**;
   - в **Team** выберите свою команду (личный Apple ID).
4. Вверху Xcode выберите схему **BlurAdaptiveCamera** и ваш **iPhone** (не симулятор — в симуляторе нет реальной камеры).
5. Нажмите **Run** (▶). При первом запуске на iPhone: **Настройки → Основные → VPN и управление устройством** — при необходимости доверьте сертификат разработчика.

При желании смените **Bundle Identifier** (`com.example.BlurAdaptiveCamera`) на уникальный (например `имя.bluradaptive`).

## Что показать на защите

1. **Теплокарта**: краснее там, где ниже локальная «энергия» краёв (условно размытее); зеленее — резче.
2. Индикатор **Blur** и число **Var** (медиана дисперсии Лапласа по блокам — условные единицы, зависят от масштаба анализа).
3. Переключатель **«Коррекция резкости»**: видно усиление резкости постобработкой при более размытой сцене (в разумных пределах; сильный **дефокус / смаз** физически не восстанавливается).
4. **«Рамки лиц (Vision)»**: детекция лиц как элемент «локализации объектов».
5. Поле **«Анализ WxH»**: при росте нагрузки разрешение даунскейла для метрики может **снижаться**, при запасе по времени — **расти** (адаптивное качество конвейера анализа).

## Структура исходников

| Файл | Назначение |
|------|------------|
| `CameraSession.swift` | `AVCaptureSession`, `AVCaptureVideoDataOutput`, ориентация для Vision/CI |
| `BlurAnalyzer.swift` | даунскейл, grayscale, Лаплас по сетке, heatmap и глобальный скор |
| `SharpenPipeline.swift` | Core Image `CISharpenLuminance` |
| `FaceOverlayDetector.swift` | `VNDetectFaceRectanglesRequest` с ограничением частоты |
| `CameraViewModel.swift` | очередь обработки, сглаживание параметров, FPS |
| `ContentView.swift` | превью, HUD, переключатели |

## Замечание для отчёта

В задании упоминаются OpenCV / MediaPipe — здесь сознательно использован **нативный стек Apple** (проще сборка и установка на устройство). Алгоритмическая идея та же: **Лапласиан / дисперсия** как метрика фокуса, **адаптивная** постобработка резкости, **детекция** через Vision.
